OS2 Final Lab Exam Readme
CS16BTECH11038 Supreet Singh
=============================

Compilation
-----------
Compile  the programs with:
    $ g++ cs16btech11038.cpp -pthread -std=c++11 
    

Execution
---------
The programm takes in input from file "inp-params.txt". The input is SEVEN (7) space separated 
integers, as mentioned in the assignment.

Execute the compiled program (in the directory with "inp-params.txt") with: 
    $ ./a.out

Output
------
The program will create the following output files: log.txt for the log of all events,
and Average_time.txt for the average waiting times of each tribesman.
